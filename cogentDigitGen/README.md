##### Base library
These are base library files. Each directory explanation is mentioned below:

```
./data/t10k-images.idx3-ubyte               -> Default Image File
./data/t10k-labels.idx1-ubyte               -> Default Label File
./etc/config.ini                            -> Default Config File
./lib/cogentConfigParser.py                 -> Config Parser Class
./lib/cogentDigitGenerateReadImageData.py   -> Default Reader Class
./lib/cogentDigitGenerateSequence.py        -> Main Control file for image generator
./lib/cogentLogging.py                      -> Default Logging Class
./lib/cogentUtils.py                        -> Common utilities class
./lib/__init__.py                           -> Python Package file
```
